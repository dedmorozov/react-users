# react-users app

[DEMO LINK](https://vamonospest.github.io/react-users/)

## Local deployment

### Dependencies
* Node v14.7.0
* NPM v6.14.8
* eslint v7.13.0
* node-sass v4.0.0
* prop-types v15.7.2

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
